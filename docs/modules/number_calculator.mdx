## Cài đặt

Hỗ trợ các phép tính toán:
```js

+ (phép cộng)
- (phép chia)
* (phép nhân)
/ (phép chia)
% (phép chia lấy phần dư): ví dụ 5%3 = 2.
^ (lấy mũ): ví dụ 3^2 = 9.
```
Hỗ trợ các hàm nâng cao:

```js
! (giai thừa): ví dụ 5! = 120.
log (hàm log): ví dụ '2 log 10' = 0.301029995663981.
max (hàm max): ví dụ '3 max 2' = 3.
min (hàm min): ví dụ '3 max 2' = 2.
sqrt (căn bậc 2): ví dụ sqrt(9) = 3.
sin (hàm sin): ví dụ sin(3.14/2) = 1. Đơn vị theo radian.
cos (hàm cos): ví dụ cos(3.14/2) = 0. Đơn vị theo radian.
tan (hàm tan): ví dụ tan(3.14) = 0. Đơn vị theo radian
ceil (phép làm tròn lên): ví dụ ceil(3.2) = 4.
floor (phép làm tròn xuống): ví dụ floor(3.7) = 3.
round (phép làm tròn): ví dụ round(3.2) = 3, round(3.6) = 4.
exp (hàm e mũ): ví dụ exp(2) = 7.38905609893065.
abs (hàm lấy giá trị tuyệt đối): ví dụ abs(-6) = 6.

```

Ví dụ 1: kết quả là `23`.

```js
(3 + 2) * 5 - sqrt(4)
```

Ví dụ 2: kết quả là `4`.

```js
8%3 + (2 max 1)
```

Ví dụ 3: kết quả là `-19`.

```js
abs(-6) + sin(3.141592653589793238462643) - 5^2
```
