## Cài đặt

Các dạng điều kiện hỗ trợ:
-   **True/False**: dữ liệu so sánh dạng true/false, lựa chọn dữ liệu này từ các module đằng trước. Hỗ trợ kiểu so sánh:
    -   **Equal**: 2 dữ liệu giống nhau sẽ được kết luận là đúng (True).
    -   **Not Equal**: 2 dữ liệu này trái ngược nhau sẽ được kết luận là đúng (True).
-   **Dạng số**: dữ liệu so sánh đầu vào dạng số, hỗ trợ các kiểu so sánh só học: `=` `!=` `<` `<=` `>` `>=`.
-   **Dạng chữ**: dữ liệu so sánh dạng chữ, hỗ trợ kiểu so sánh:
    -   **Equal**: 2 dữ liệu giống nhau sẽ được kết luận là đúng (True).
    -   **Not Equal**: 2 dữ liệu này trái ngược nhau sẽ được kết luận là đúng (True).
    -   **Contain**: nếu dữ liệu đầu tiên chứa dữ liệu thứ 2 sẽ được kết luận là đúng (True).
    -   **Not Contain**: nếu dữ liệu đầu tiên **không** chứa dữ liệu thứ 2 sẽ được kết luận là đúng (True).
