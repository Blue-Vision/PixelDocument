## Thông số cài đặt

-   **Loại code**: chọn loại 2D code cần đọc.
-   **Tham số mặc định**: có 3 thông số từ cơ bản tới nâng cao. Tham số nâng cao sẽ đọc được 2D code trong nhiều trường hợp chất lượng hình ảnh của code không tốn, nhưng cũng tốn nhiều thời gian đọc hơn.
    -   **standard_recognition**: chế độ cơ bản.
    -   **enhenced_recognition**: chế độ nâng cao.
    -   **maximum_recognition**: chế độ tối đa.
-   **Phân cực**: chọn chế độ màu sắc in của 2D code so với nền. Với mã 2D code màu đen nền trắng cần chọn **dark_on_light**,
với 2D code màu trắng nền đen chọn **light_on_dark**.

## Cài đặt giới thạn
Các giới hạn cài đặt dựa trên kết quả:
-   **So sánh nội dung**: kết quả công cụ đúng nếu nội dung đọc được với nội dung người dùng nhập giống nhau.
-   **Kiểm tra nội dung có trong kết quả**: kết quả công cụ đúng nếu nội dung đọc được chứa nội dung người dùng nhập.
-   **So sánh độ dài mã**: kết quả công cụ đúng nếu độ dài nội dung đọc được với độ dài người dùng nhập giống nhau.
