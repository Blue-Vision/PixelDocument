# Variable

Phần mềm cho phép người dùng tạo ra biến để có thể sử dụng trong toàn bộ chương trình. Hỗ trợ biến dạng chữ, dạng số, dạng True/False và dạng ảnh.

**Ứng dụng phổ biến:**
- :white_check_mark: Hỗ trợ tốt cho những logic phức tạp cần sử dụng biến.

## Cài đặt

Các dạng biến hỗ trợ:

-   **Dạng số**: số dạng thập phân (double).
-   **Dạng chữ**.
-   **Dạng Boolean**: True/false.
-   **Dạng ảnh**.
