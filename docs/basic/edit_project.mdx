# Cài đặt Project

## Tạo mới
Để tạo chương trình mới người dùng mở phần mềm `Pixel > Tạo Chương Trình`.

<Image src="/images/basic/create_project.png" width='600px' />

Nếu như người dùng đang ở trang cài đặt thì có thể lựa chọn tạo chương trình trong thanh công cụ `Tạo mới`.

<Image src="/images/basic/create_project_2.png" width='600px' />

## Chỉnh sửa
Để chỉnh sửa project người dùng mở phần mềm `Pixel > Chọn project cần chỉnh sửa`. Chú ý nhấn vào biểu tượng ![Edit project](/images/basic/icon_pencil.png) để chỉnh sửa.

<Image src="/images/basic/edit_project.png" width='600px' />

Hoặc người dùng có thể chọn lựa project trong danh sách project.

<Image src="/images/basic/edit_project_2.png" width='600px' />

Sau khi chỉnh sửa xong project người dùng nhấn lưu lại project.

<Image src="/images/basic/save_project.png" width='600px' />

## Xóa

Để xóa project vào `công cụ > chọn xóa project`.

<Image src="/images/basic/remove_project.png" width='600px' />
